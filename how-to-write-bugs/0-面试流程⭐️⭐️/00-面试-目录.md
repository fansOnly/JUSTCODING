- Webpack 是如何打包的
  1. 搭建架构，读取配置参数
  2. 根据配置参数初始化 Compiler 对象
  3. 挂载配置文件中的插件
  4. 执行 Compiler 对象的 run 方法开始编译
  5. 根据配置文件的 entry 找到所有的入口
  6. 根据入口文件，调用 loader 对模块进行编译
  7. 递归编译所有模块的依赖模块
  8. 编译完所有模块后，根据依赖关系，生成代码 chunk
  9. 将所有的 chunk 转换成输出文件
  10. 根据配置的出口和文件名，讲文件输出到文件系统
