### 主要技术栈

- **Javascript 基础** - HTML5 + CSS3 + jQuery
- **Javascript 进阶** - Canvas + Animation + Echarts
- **Vue 全家桶** - Vue2/3 + vuex + vue-router + Axios + Element / Vant
- **小程序** - uni-app
- **打包工具** - Vue-CLI / Webpack / Vite
- **包管理器** - Node / Yarn / Pnpm
- **版本管理** - Git / Svn
